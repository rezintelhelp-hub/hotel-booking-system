üìù Content & Pages

GET /api/public/client/{clientId}/site-config Public Get full site configuration

Returns complete site configuration including branding, contact info, pages, navigation, and more. Used by WordPress themes.

Response includes:
  • contact - Business name, email, phone, address, social links
  • branding - Colors, fonts, logo, header/footer styles
  • pages - About, Contact, Terms, Privacy content
  • navigation - Header and footer menu items
  • properties - Property list with room counts
  • features - Feature flags (has_blog, has_attractions, etc.)
  • website - Website builder settings (hero, intro, CTA, etc.)
GET /api/public/client/{clientId}/page/{pageType} Public Get single page content

Returns content for a specific page type: about, contact, terms, privacy, faq

GET /api/public/client/{clientId}/blog Public Get blog posts
ParameterTypeDescription
category string Filter by category
limit integer Number of posts (default: 10)
offset integer Pagination offset

üîî Webhooks

POST /api/webhooks/smoobu Public Smoobu webhook receiver

Receives webhook events from Smoobu for booking updates, cancellations, and modifications.

Events Supported:
  • newReservation - New booking created
  • cancelReservation - Booking cancelled
  • updateReservation - Booking modified
POST /api/webhooks/beds24 Public Beds24 webhook receiver

Receives webhook events from Beds24 for real-time booking sync.

POST /api/webhooks/hostaway Public Hostaway webhook receiver

Receives webhook events from Hostaway for real-time booking sync.
